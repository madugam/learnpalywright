# LearnPlaywright

A beginner-friendly repository for learning **Playwright** automation testing and **JavaScript** fundamentals, created by **Narsimha**.

---

## About

This repository documents my learning journey with Playwright and JavaScript. It contains practical examples, code snippets, and reference materials organized by chapters.

---

## Repository Structure

```
LearnPlaywright/
├── Chapter_01_Basics/
│   ├── 01_Basics.js          # Hello World & basic console logging
│   ├── 02_Num.js             # Number operations and examples
│   └── 03_JS_setupCheck.js   # JavaScript environment setup verification
│
└── Chapter_02_JavaScriptBasics/
    ├── comments.js           # Keyboard shortcuts reference (Windows & Mac)
    ├── IndentifierRules.js   # Naming conventions: camelCase, PascalCase, snake_case, etc.
    ├── JsBasics.js           # JavaScript fundamentals
    └── JsCases.js            # Examples of different naming cases in JS
```

---

## Chapters Overview

### Chapter 01: Basics
Introduction to JavaScript basics including:
- Console output (`console.log`)
- Number operations
- Environment setup and checks

### Chapter 02: JavaScript Basics
Core JavaScript concepts including:
- **Identifier Rules** — Naming conventions (camelCase, PascalCase, snake_case, UPPER_SNAKE_CASE, kebab-case, Train-Case, dot.case)
- **Keyboard Shortcuts** — Reference for Windows & Mac keyboard keys with JavaScript `KeyboardEvent` properties
- **JS Cases** — Practical examples of different naming conventions used in real code

---

## How to Run JavaScript Files

### Using Node.js
```bash
node Chapter_01_Basics/01_Basics.js
```

### In a Browser
Open your browser console (`F12` → Console tab) and paste the code, or include the script in an HTML file:
```html
<script src="Chapter_01_Basics/01_Basics.js"></script>
```
